## Getting Started

### Usage

``` bash
clone repo 
npm install
npm run dev
```

### Deploy


1. 编辑 config/index.js，修改第 10 行的 assetsPublicPath，值为 `项目名/dist`。如果你没有修改项目名 ，则可跳过此步骤。

2. 编译、上传
    ``` bash
    npm run build
    git add .
    git commit -m {Update-Info}
    git push
    ```

3. 开启 GitHub Pages 功能
